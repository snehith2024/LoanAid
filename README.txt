LoanManagerApp (KivyMD Version)
===============================

A simple loan registration Android-style app built using KivyMD (Material Design for Kivy).

Features:
- Add, view, edit, and delete loan records
- Material Design interface with clean UI
- Local data saved in loans.json

How to Run:
-----------
1. Make sure you have Python 3.10+ and install KivyMD:

   pip install kivymd

2. Run the app:

   python main.py
